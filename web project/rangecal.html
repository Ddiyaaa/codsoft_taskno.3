Range Formula Description

The range is a basic statistical measure that provides information about the spread or dispersion of a dataset. It is the simplest way to understand the variability between the largest and smallest values in a dataset, making it a useful measure for quick data assessment.

Range Formula

The range (R) of a dataset x₁, x₂, x₃, ..., xₙ can be calculated using the following formula:

R = Maximum Value - Minimum Value

Where:

  • R represents the range of the dataset.
  • The "Maximum Value" is the largest value in the dataset.
  • The "Minimum Value" is the smallest value in the dataset.
